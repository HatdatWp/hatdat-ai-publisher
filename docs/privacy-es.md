# Hatdat AI Publisher — Aviso de privacidad y API de OpenAI

Hatdat AI Publisher utiliza la API de OpenAI para generar textos de artículos, prompts de imagen, imágenes y metadatos SEO.

## Datos enviados a OpenAI

Solo se envía a OpenAI el contenido que un administrador envía intencionadamente desde la interfaz de administración de Hatdat AI Publisher. Esto puede incluir prompts, URL de origen, temas de artículos, borradores, solicitudes de generación de imágenes e instrucciones editoriales.

## Datos no enviados automáticamente

Hatdat AI Publisher no transmite automáticamente a OpenAI cuentas de usuario de WordPress, contraseñas, direcciones de correo electrónico, datos de perfil u otros registros de usuarios de WordPress.

## Costes

El uso de Hatdat AI Publisher requiere una cuenta API de OpenAI independiente. El uso de la API puede generar costes adicionales facturados por OpenAI. La generación de imágenes puede representar una parte significativa del coste total.

## Responsabilidad

Los administradores del sitio son responsables de garantizar que el contenido enviado a OpenAI cumpla con las leyes aplicables de privacidad, derechos de autor y protección de datos.
