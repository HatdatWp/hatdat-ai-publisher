# Hatdat AI Publisher Privacy and OpenAI API Notice

Hatdat AI Publisher uses the OpenAI API to generate article text, image prompts, images and SEO metadata.

## Data sent to OpenAI

Only content intentionally submitted by an administrator through the Hatdat AI Publisher administration interface is sent to OpenAI. This may include prompts, source URLs, article topics, draft article text, image generation requests and related editorial instructions.

## Data not automatically sent

Hatdat AI Publisher does not automatically transmit WordPress user accounts, passwords, email addresses, profile data or other WordPress user records to OpenAI.

## Costs

Using Hatdat AI Publisher requires a separate OpenAI API account. API usage may generate additional costs billed by OpenAI. Image generation can cause a significant part of the total cost.

## Responsibility

Site administrators are responsible for ensuring that submitted content complies with applicable privacy, copyright and data protection laws.
